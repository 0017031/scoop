    A command-line installer for Windows

    Scoop installs the tools and utilities you need

    PS> scoop install curl


    Get comfortable on the Windows command line

    Whether you're looking for familiar tools from other operating systems or you just want alternatives to the Powershell Verb-Noun way of doing things, Scoop helps you get the programs you need, with a minimum amount of point-and-clicking.


    Installs in seconds

    Make sure Powershell 3 is installed, then run:

    iex (new-object net.webclient).downloadstring('https://get.scoop.sh')

    Note: if you get an error you might need to change the execution policy (i.e. enable Powershell) with set-executionpolicy unrestricted -s cu
